# 🏧 ATM System Simulation

A console-based ATM simulation built with **C++** that allows users to:

- ✅ Verify PIN (3 attempts before lock)
- ✅ Check account balance
- ✅ Deposit money (max 5000 per transaction)
- ✅ Withdraw money (with balance check)
- ✅ View mini statement (last transaction)
- ✅ Exit safely

## 🛠️ Technologies Used

- C++
- Standard Input/Output Library
